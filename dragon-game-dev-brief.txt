Dragon RPG Game — Dev Brief

Project Overview
Browser-based dragon RPG inspired by Wings of Fire.
Focus: exploration, story, light gameplay (hunting, relationships, progression).
Audience: kids (simple, visual-first UX).

Tech Stack
- Frontend: Vanilla JavaScript (no frameworks)
- Architecture: Global scripts, Scenes system
- Backend: Supabase (Postgres)
- Hosting: GitHub Pages
- Assets: Static images (/images)

Core Architecture
Game Engine (game.js)
- Game.state
- Game.setScene(), render(), handle()

Scenes
Scenes.X = { name, render, handle }

Systems
- SupabaseSystem
- InventorySystem
- RelationshipSystem

Game Systems

World Map
- DB-driven
- clickable markers
- uses map_x/map_y

Player Model
- player = dragon
- multiple dragons per user

Hunger System
- decreases with actions
- restored via hunting

Hunting System
- location-based
- choice system: wait / stalk / fly

NPC System
- loadNPCs(location_id)
- dynamic rendering
- trust system

Objective System
- state.currentObjective
